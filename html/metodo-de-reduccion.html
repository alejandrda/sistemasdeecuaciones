Sistemas de ecuaciones

Método de reducción

El método de reducción consiste en sumar o restar las ecuaciones del sistema para eliminar una de las incógnitas, obteniendo así una ecuación con una sola variable.

 
Pasos a seguir

  1. Preparar las ecuaciones, multiplicando si es necesario para que los coeficientes de una incógnita sean opuestos.
  2. Sumar o restar las ecuaciones para eliminar una incógnita.
  3. Resolver la ecuación resultante (de una sola incógnita).
  4. Sustituir el valor obtenido en una de las ecuaciones iniciales.
  5. Comprobar la solución.

Consejos y errores comunes

  • Si los coeficientes no son opuestos, multiplica las ecuaciones antes de sumar o restar.
  • Cuidado con los signos al operar.
  • Es recomendable elegir la incógnita más fácil de eliminar.
  • No olvidar comprobar la solución.

¿Cuándo usar este método?

El método de reducción es especialmente útil cuando:

  • Los coeficientes de una incógnita son iguales u opuestos.
  • Se puede eliminar fácilmente una variable sin complicar las operaciones.
